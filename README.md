# Honnou

> A practical mathematics workbook for building geometric intuition.

Honnou is a workbook and training environment for developing the instincts that
make working with advanced algebra tractable — the kind of pattern recognition
that separates fluent practitioners from those who must derive everything from
first principles every time.

## Scope

The workbook targets intuitions across several mathematical fields that recur
throughout the Industrial Algebra ecosystem:

- **Grassmannians** — recognizing useful subspaces, intersections, and
  decompositions.
- **Clifford algebra signatures** — developing feel for which signature
  R^{p,q} is the right setting for a given problem.
- **Combinatorial Game Theory (CGT)** — eventual extensions for surreal
  numbers, thermographs, and game decomposition.
- Other necessary intuitive skills, added as the curriculum matures.

The approach is **practical and interactive**, not encyclopedic: exercises,
worked patterns, and scaffolded problems that build recognition.
